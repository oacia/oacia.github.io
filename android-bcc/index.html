# bcc 环境搭建

接下来进行 bcc 的环境搭建，我们将使用 eadb 在手机中配置 Debian 系统，并通过 ssh 让 Windows 上的 vscode 可以连接手机中的 Debian

想要使用 bcc 的全部功能，首先检查 linux 内核版本，我的内核版本为 5.10

然后检查 BPF 的配置是否开启

随后使用 magisk root 手机，然后安装 MagiskSSH
