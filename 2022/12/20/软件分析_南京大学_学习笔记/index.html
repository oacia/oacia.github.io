Why We Learn Static Analysis?

  • 通过静态分析,可以提高程序的可靠性(比如避免空指针引用,内存泄漏等)
  • 提高程序的安全性(如私有信息泄露,恶意代码攻击)
  • 编译优化(如消除死代码,代码移动(将不必要的重复运行的语句优化))
  • 理解程序

What is Static Analysis?

一个有用的静态分析,最优先需要保证的是它的正确性(soundness),可以允许误报,但不能允许漏报,因为漏报一个bug,程序依旧会崩溃.

conclusion

Static Analysis:ensure (or get close to) soundness,while making good trade-offs between analysis precision and analysis speed.

6.Basic Blocks(BB)

为了更好的理解基本块(Basic Blocks)的的概念，我按照规则简单的写了一下python代码。

import re

AC3code = '''(1)x = input
(2)y = x - 1
(3)z = x * y
(4)if z < x goto (7)
(5)p = x / y
(6)q = p +y
(7)a = q
(8)b = x + a
(9)c = 2a - b
(10)if p==q goto (12)
(11)goto (3)
(12)return
'''
AC3seg = AC3code.split('\n')  # this list(AC3seg) contains every segment in AC3code
# print(AC3seg)
segIn = [False for _ in range(len(AC3seg) + 1)]
segOut = [False for _ in range(len(AC3seg) + 1)]
for i in range(len(AC3seg)):
    segment = AC3seg[i]
    if 'goto' in segment:
        tem = re.findall(r"goto [(](\d+)[)]", segment)
        # print(segment, tem)
        block_in = int(tem[0])
        segIn[block_in] = True
        segOut[i + 1] = True
Block_ = 1
seg_ = 1
while seg_ < len(AC3seg) + 1:
    if seg_ == 1:
        print("\nBasic Block%d:%d" % (Block_, seg_), end='')
    # if one segment is going to enter the current segment,then new block begins.
    # if current segment is going to the other segment by "goto" operation,then block ends.
    elif segIn[seg_] or segOut[seg_-1]:
        Block_ += 1
        print("\nBasic Block%d:%d" % (Block_, seg_), end='')
    else:
        print(", %d" % seg_, end='')
    seg_ += 1
