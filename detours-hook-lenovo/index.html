# detours 的编译

首先从 github 上把源码下载下来

随后进入 vc 文件夹，里面有个 Detours.sln 直接双击打开 (确保电脑上有 visual studio)

这里切换成 Release , x64 , 然后按下 Ctrl + B 生成 Detours

编译完成之后会在根目录生成这两个文件夹，里面有我们 hook 需要的 detours.h 和 detours.lib
